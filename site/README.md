# Teamsly Website

Static marketing site for Teamsly. No build step, no dependencies.

Open `index.html` directly in a browser or serve the directory with any static
host (GitHub Pages, Vercel, S3, etc.).

Files:

- `index.html` — page structure and copy.
- `styles.css` — visual system and responsive layout.
- `logo.svg` — standalone brand mark (also inlined in the page for header,
  footer, and favicon).
